tTable.js
=========

tTable.js - plugin for work with table data.

[@MacPaw](http://macpaw.com) FedEX Day project
=========

Features
=========
 - Rows numbers;
 - Navigation Arrows;
 - Pagination;
 - GoTo page;
 - Page size;
 - Column sorting;
 - Count totals;
